# RMI Weather Monitor - Client

RMI Weather Monitor Client is the frontend component of the JavaFX-based weather monitoring system. The Client receives data from the Server via RMI and provides an interactive user experience.

## Key Features
- Receive real-time, multi-day weather forecasts from the Server.  
- Display data on **LineCharts for temperature** and **BarCharts for rainfall**.  
- **View and edit user profiles**.  
- **Admin/Co-Admin user management**, including viewing the list, editing, and statistics.  
- Display a **LineChart of user registrations over time**.  
- Display a **PieChart showing the days with the most registrations** to identify peak user sign-up periods.  
- Support UDP chat, disaster alerts, and automatic login emails.
